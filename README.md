# TP1 Prog III — Sitio Web (HTML + CSS)

Proyecto de sitio web multipágina hecho con HTML semántico y CSS externo.

## Integrantes

- Agustin Gonzalez
- Juan Manuel Aguilar

## Qué contiene el código?

- `index.html`: página principal (Home) con presentación y accesos rápidos.
- `servicios.html`: detalle de los 3 servicios (realizados por Agustin Gonzalez y Juan Manuel Aguilar).
- `equipo.html`: información del equipo.
- `pedido.html`: formulario para pedir un servicio.
- `contacto.html`: formulario de contacto.
- `faq.html`: preguntas frecuentes.
- `css/styles.css`: estilos generales del sitio (tipografías, colores, layout, cards, botones y formularios).
- `assets/`: recursos del sitio (imágenes/íconos en `assets/img`).

## Cómo está organizado?

- Todas las páginas comparten una estructura similar: `header` (barra de navegación), `main` (contenido) y `footer`.
- La navegación funciona con links relativos entre los `.html`.
- Los estilos se cargan en todas las páginas con `<link rel="stylesheet" href="./css/styles.css" />`.

## Cómo ejecutar

Abrí `index.html` con doble click (o “Abrir con” tu navegador).
