Preguntas frecuentes

Respuestas rápidas sobre cómo trabajamos y qué incluye cada servicio.

¿Qué incluye el servicio de Desarrollo Web?

Incluye páginas conectadas entre sí, estructura semántica y CSS externo con estilos consistentes.

¿Puedo pedir un diseño responsive?

Sí. Se puede solicitar como extra para adaptar el layout a pantallas chicas (mobile-first).

¿Cómo se organiza el trabajo en equipo?

Se trabaja con Git y GitHub usando ramas (main, dev y una por integrante) y pull requests.

¿Qué datos necesito para iniciar un pedido?

Nombre, email/teléfono y el servicio elegido. Luego se detallan objetivos y fecha deseada.

¿Cómo los contacto si tengo dudas?

Podés usar el formulario de Contacto o iniciar un pedido desde Pedido.
